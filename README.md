# Tuk-Tuk Tracking API

RESTful API for Real-Time Three-Wheeler (Tuk-Tuk) Tracking & Movement Logging System for Law Enforcement in Sri Lanka.

## Student ID

COBSCCOMP242P-055

## Description

This API provides endpoints for managing vehicles, drivers, administrative boundaries, and secure access for authorized users.

## Installation

1. Clone the repository
2. Run `npm install`
3. Run `npm start`
